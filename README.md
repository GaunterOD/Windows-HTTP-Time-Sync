# Windows HTTP Time Synchronizer
> **NTP(UDP 123) 포트 차단 환경을 위한 시간 동기화 도구**

이 프로젝트는 LG 유플러스(LGU+) 등 특정 ISP의 회선 방화벽이나 사내 보안망, 기숙사 네트워크에서 표준 NTP 포트(UDP 123)를 차단하여 윈도우 시간 동기화가 불가능한 문제를 해결하기 위해 제작되었습니다.

---

## 문제 증상 (Problem Symptoms)

PC 시간이 실제 표준 시간과 맞지 않고 아래와 같은 현상이 발생한다면 네트워크 회선에 의한 차단을 의심해야 합니다.

1. **설정창 무한 로딩**: 윈도우 설정의 [지금 동기화] 버튼을 눌러도 응답이 없거나 무한 로딩이 지속됨.
2. **동기화 에러 발생**: 제어판 고급 시간 설정에서 동기화 시도 시 아래 메시지 출력과 함께 실패함.
   > *"로컬 컴퓨터에 다시 동기화 명령을 보내는 중 사용 가능한 시간 데이터가 없어 컴퓨터가 동기화하지 못했습니다."*
3. **서비스 연쇄 오류**: 시간 불일치로 인한 거래소 API(Binance, BitMEX 등) 연결 실패, 웹사이트 SSL 인증서 에러 발생.

## 해결 방식 (The Solution)

본 도구는 방화벽에 막힌 UDP 123 포트 대신, 일반적인 웹 통신 경로인 **HTTPS (TCP 443 포트)**를 이용합니다. Google 서버의 응답 헤더에서 표준 시간 데이터를 추출하여 시스템 시계를 강제 동기화하므로, 인터넷 접속만 가능하다면 어떤 환경에서도 정확한 시간 설정이 가능합니다.

---

## 주요 기능 (Key Features)

* **NTP 포트 우회**: HTTP/HTTPS 기반으로 통신하여 ISP 및 사내 방화벽 제약을 무력화합니다.
* **독립 실행 (Standalone)**: 파이썬이 설치되지 않은 PC에서도 `.exe` 파일 하나로 즉시 실행 가능합니다.
* **권한 자동 승인**: 실행 시 시스템 시간 변경에 필요한 관리자 권한(UAC)을 자동으로 요청합니다.
* **프로세스 유지**: 동기화 완료 후 엔터 입력 전까지 창이 닫히지 않아 결과 확인이 용이합니다.

## 사용 방법 (How to Use)

### 방법 1. 실행 파일(.exe) 사용 (일반 사용자 권장)
파이썬 설치가 필요 없는 가장 간단한 방법입니다.
1. 저장소의 `sync_time.exe` 파일을 다운로드합니다.
2. 다운로드한 `sync_time.exe`를 **더블 클릭**하여 실행합니다.
3. 관리자 권한 승인 창이 뜨면 **[예]**를 선택합니다.
4. 동기화 완료 메시지 확인 후 엔터를 눌러 종료합니다.

### 방법 2. 파이썬 스크립트(.py) 사용 (개발자용)
코드를 직접 확인하고 환경에 맞춰 실행하고 싶은 경우 사용합니다.
1. Python 3.x가 설치된 환경에서 필수 라이브러리를 설치합니다.
   ```bash
   pip install requests
   ```
2. `sync_time.py` 파일을 다운로드합니다.
3. 명령 프롬프트(CMD) 등에서 파이썬 파일을 직접 실행합니다. (실행 시 자동으로 관리자 권한을 요청합니다.)
   ```bash
   python sync_time.py
   ```
